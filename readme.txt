== Description ==

Easy way to allow your site members to post and comment anonymously while logged in. It adds a checkbox to the 'Add Post' form, which allows to remove the author name when publishing the post. It also displays a 'Post Anonymously' checkbox in the comment form so that your blog member can choose to comment as 'anonymous'. Easy to use, and good for its purpose.

== Installation ==

Follow the below steps to install the plugin:
1. Unzip the plugin archive and upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress dashboard.
3. Check the 'Post Anonymously' box when adding or editing a post to publish it anonymously.
4. Check the 'Post Anonymously' box when adding a comment to publish it anonymously.
